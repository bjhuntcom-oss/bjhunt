05

Usage contract

Where each variant lives

Solid

The default. Use for single-item emphasis: FindingCard header, hero stat, notification. Pulse reserved for critical severity on live stream.

Outline

Use for filter pills, dashboard tiles, and anywhere the badge coexists with dense color. Softer presence, preserves contrast on bright backgrounds.

Minimal

Use for tables, lists, and rows where the column itself communicates "severity". Just the dot + word. No chrome.

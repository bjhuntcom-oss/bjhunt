# BJHUNT engine patches

Tracked overrides on top of the upstream PurpleAILAB/Decepticon engine.
Each patch documents what was changed in `engine/decepticon/` so it can be
re-applied after `ops/scripts/sync-upstream-decepticon.sh` pulls a new
release. Per CLAUDE.md D3 + D7 we keep the upstream code intact and ship
patches here.

Naming: `NNNN-short-slug.md` (one per logical change).
